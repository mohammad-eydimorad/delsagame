# دنیای بازی دلسا و بابا

این پروژه یک Progressive Web App است و پس از اولین بارگذاری، بازی‌ها بدون اینترنت نیز اجرا می‌شوند.

فونت فارسی Vazirmatn به‌صورت محلی از پوشه `fonts` بارگذاری می‌شود و مجوز OFL آن نیز داخل همان پوشه قرار دارد.

## اجرای محلی

Service Worker روی `file://` اجرا نمی‌شود. پروژه را با یک وب‌سرور محلی باز کنید:

```powershell
python -m http.server 8000
```

سپس آدرس زیر را باز کنید:

```text
http://localhost:8000
```

برای انتشار عمومی، سایت باید روی HTTPS میزبانی شود.

## به‌روزرسانی کش

بعد از تغییر فایل‌های کش‌شده، مقدار `CACHE_NAME` در `sw.js` را افزایش دهید؛ برای مثال:

```js
const CACHE_NAME = "delsa-game-v3";
```
